# Support

## Getting Help

- Usage and setup: open a GitHub Discussion (recommended for questions).
- Bugs and regressions: open a GitHub Issue using the bug template.
- Security concerns: follow `SECURITY.md`.

## Before Opening An Issue

- Review `README.md` and `docs/DEVELOPMENT.md`.
- Confirm your issue reproduces on the latest alpha.
- Include platform details (OS, app version, steps to reproduce).
